# WinGuard

## Descripción

WinGuard es un antivirus para entornos Windows (De momento). Ofrece detección y limpieza de amenazas, protección en tiempo real y utilidades para mejorar la seguridad del sistema.

## Contenido

- Motor de detección de malware
- Herramientas de limpieza y análisis
- Documentación básica para uso e instalación

## Requisitos

- Windows 10 o superior
- PowerShell 5.1 o posterior
- Dependencias específicas según cada componente

## Instalación

1. Clona el repositorio:

   ```powershell
   git clone https://github.com/tu-usuario/winguard.git
   ```

2. Navega al directorio del proyecto:

   ```powershell
   cd winguard
   ```

3. Revisa la documentación de cada componente antes de ejecutarlo.

## Uso

- Abre PowerShell con permisos de administrador cuando sea necesario.
- Ejecuta las utilidades de WinGuard directamente o importa las funciones según la necesidad.
- Consulta los comentarios dentro de cada archivo para conocer los parámetros disponibles.
